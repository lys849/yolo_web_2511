- 总体目标：

    用户在网页上传一张图片-后端调用yolo模型识别图片内容-返回识别结果-网页展示识别结果

    前端网页 - 后端api服务 - 模型推理

- 总体架构：

    - 前端网页：
        - 文件上传按钮
        - 发送图片到后端借口
        - 接受识别结果
        - 显示结果
    - 后端：
        - 接受前端上传的图片
        - 调用yolo模型进行推理
        - 生成结果图
        - 返回响应结果
    - 模型：
        - 加载模型文件
        - 进行目标检测推理
        - 输出
